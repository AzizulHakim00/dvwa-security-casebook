02 WORKING METHOD

REPEATABLE PROCESS

From hypothesis to defensible finding.

The method is designed so a reviewer can understand the result without recreating every lab action.

01

Scope

Define target, authorization, security level, boundary, and expected secure behavior.

02

Probe

Use focused browser, proxy, and command-line inputs against one control at a time.

03

Capture

Preserve request context and observable output as terminal and report evidence.

04

Remediate

Connect root cause to practical developer, platform, and monitoring controls.
